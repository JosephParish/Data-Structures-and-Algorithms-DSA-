# Algorithms & Data Structures (Java)

A personal repository of Java implementations for common algorithms and data structures.

## Purpose

This project serves as a practical way to:

- Practice coding consistently
- Strengthen understanding of core computer science concepts
- Explore and implement new algorithms
- Build a reference library for interview preparation and learning

## Implemented Topics

### Data Structures
- Stack
- Queue
- Linked List
- Heap
- Binary Tree
- Tree
- Blockchain (experimental)

### Search Algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- A* Search
- Uniform Cost Search (UCS)
- Greedy Search (GS)
- Hill Search (HS)
- Minimax

### Sorting Algorithms
**Comparison Sorts**
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Cycle Sort
- Three-Way Merge Sort

**Hybrid Sorts**
- IntroSort
- TimSort

**Non-Comparison Sorts**
- Counting Sort
- Radix Sort
- Bucket Sort
- Pigeonhole Sort
- Comb Sort
- Bogo Sort

## Future Work

Additional algorithms, optimisations, tests, and documentation will be added over time.
